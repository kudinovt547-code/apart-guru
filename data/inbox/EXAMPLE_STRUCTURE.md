# Структура данных для импорта

## Excel файл (*.xlsx)

Первая строка должна содержать заголовки колонок. Ожидаемые колонки:

| Колонка | Описание | Тип | Обязательно |
|---------|----------|-----|-------------|
| `title` | Название объекта | string | Да |
| `city` | Город | string | Да |
| `country` | Страна | string | Да |
| `format` | Формат (apartment/hotel/hostel) | string | Нет (по умолчанию: apartment) |
| `status` | Статус (active/planning) | string | Нет (по умолчанию: active) |
| `price` | Стоимость в рублях | number | Да |
| `area` | Площадь в м² | number | Да |
| `revPerM2Month` | Доходность ₽/м²/мес | number | Да |
| `noiYear` | NOI в год (₽) | number | Да |
| `paybackYears` | Окупаемость в годах | number | Да |
| `occupancy` | Загрузка (%) | number | Да |
| `adr` | Средний чек за сутки (₽) | number | Да |
| `riskLevel` | Уровень риска (low/medium/high) | string | Нет (по умолчанию: medium) |
| `summary` | Краткое описание | string | Нет |
| `seasonality` | Сезонность (12 значений через запятую) | string | Нет |

## Значения полей

### format
- `apartment` - Апартаменты
- `hotel` - Гостиница
- `hostel` - Хостел
- `villa` - Вилла

### status
- `active` - Активный
- `planning` - В планировании

### riskLevel
- `low` - Низкий риск
- `medium` - Средний риск
- `high` - Высокий риск

### seasonality
Массив из 12 чисел (0-100), представляющих процент загрузки по месяцам:
- Январь, Февраль, Март, Апрель, Май, Июнь, Июль, Август, Сентябрь, Октябрь, Ноябрь, Декабрь

Если не указано, используется значение по умолчанию: `[75, 75, 75, 75, 75, 75, 75, 75, 75, 75, 75, 75]`
